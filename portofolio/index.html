Tentang Saya

Perkenalkan, nama saya Putu Gede Purnaditya Putra, biasa dipanggil Purna. Saya tinggal di Ubung Kaja, tepatnya di Jalan Irawan, dan saat ini saya menempuh pendidikan di SMK TI Bali Global Denpasar, jurusan Rekayasa Perangkat Lunak (RPL).

Di luar sekolah, saya juga aktif mengikuti kursus coding di Timedoor Coding Academy, di mana saya mempelajari HTML, CSS, dan JavaScript, serta pada tahun pertama saya belajar membuat game menggunakan Node.js. Di sekolah, saya mendalami pemrograman C++ dan juga desain UI/UX, yang menambah wawasan saya dalam membangun aplikasi dan website yang fungsional dan menarik.

Saya sangat antusias dalam mempelajari hal-hal baru, khususnya di bidang teknologi yang terus berkembang setiap harinya. Saya memiliki ketertarikan besar dalam dunia pengembangan web, terutama dalam menciptakan tampilan yang responsif, nyaman digunakan, dan menarik secara visual dengan menggabungkan prinsip UI/UX desain yang baik.

Bagi saya, teknologi bukan sekadar alat, tetapi juga media untuk menciptakan solusi yang berguna dan berdampak. Oleh karena itu, saya terus berusaha untuk berkembang dan siap menghadapi tantangan di dunia industri digital.

Dan ini beberapa project yang saya buat, saya tampilkan 3 saja UI/UX Desain, Website Developer , Dan program kasir sederhana menggunakan c++
